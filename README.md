### Run project

## Build app

Install project dependencies

> `yarn`

Install iOS dependencies to run on the iOS simulator:

> `npx pod-install`

Alternatively run:

> `yarn && npx pod-install`

For a clean install run:

> `yarn clean-install`

## Run iOS app

> `yarn ios`

## Run Android app

> `yarn android`

## Run unit test

> `yarn test`
